# 🇺🇸 Patriotic USA Website

A modern, responsive website celebrating American heritage, values, and spirit.

## Features

✨ **Modern Design**
- Clean, patriotic color scheme using official American flag colors
- Smooth animations and transitions
- Fully responsive on all devices

🎨 **Visual Elements**
- Twinkling stars background in hero section
- Color-striped flag design elements
- Gradient backgrounds with Red, White, and Blue
- Interactive hover effects on cards

📱 **Responsive Layout**
- Mobile-optimized design
- Tablet and desktop support
- Flexible grid layouts

## Sections

### 1. Navigation
Sticky navigation bar with smooth scroll links and hover effects

### 2. Hero Section
Eye-catching header with animated flag stripes and twinkling stars

### 3. American Heritage
Four heritage cards highlighting:
- Constitution
- Liberty
- Unity
- Strength

### 4. American Values
Four core values presented with numbered indicators:
- Freedom
- Justice
- Opportunity
- Community

### 5. Footer
Contact section with social media links

## Color Palette

- **Red**: #B22234 (USA Red)
- **Blue**: #3C3B6B (USA Blue)
- **White**: #FFFFFF
- **Dark Gray**: #333333
- **Light Gray**: #f5f5f5

## Technologies Used

- HTML5
- CSS3 (Animations, Flexbox, CSS Grid)
- Responsive Design with Media Queries

## How to Use

1. Copy `index.html` and `styles.css` to your repository
2. Ensure GitHub Pages is enabled in repository settings
3. Access your site at `https://[username].github.io`

## Customization

Feel free to:
- Modify content and text
- Adjust colors by changing CSS variables
- Add more sections or cards
- Include images or additional graphics

## Browser Support

Works on all modern browsers:
- Chrome/Edge (latest)
- Firefox (latest)
- Safari (latest)
- Mobile browsers
